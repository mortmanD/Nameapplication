AKAN Naming Culture

Akan names

Most people do know when their birthdays are, but some might not know what day of the week they were born.
However, Akan names are derived from Ghanian culture. Frequently in Ghana, children are given their first name as a 'day name' which corresponds to the day in the week they were born
Here are Ghanian day names

DAY Gender
Male Female
Sunday Kwasi Akosua
Monday Kwadwo Adwoa
Tuesday Kwabena Abenaa
Wednsday Kwaku Akua
Thursday Yaw Yaa
Friday Kofi Afua
Saturday Kwame Ama

About our Application

what it does

Our web application takes a user's birthday and calculates the day of the week they were born and then depending on their gender outputs their Akan Name

Items it validates

The application checks whether the date and month entered is valid. if either of them is invalid, the user should be alerted and asked to enter a valid one

The output

The application outputs to the user their Akan name depending on their gender. For example, if a user is male and the result of the calculation is 0, then the application should match that with Kwasi since Kwasi corresponds to Sunday which has an index 0. Hence the output the user will see should be something like “Your Akan name is Kwasi”
